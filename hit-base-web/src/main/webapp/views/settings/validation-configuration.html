Customize validation detections
Detection Description Classification
Structure based detections
r-usage Raised when an element with R usage is not populated
x-usage Raised when an element with X usage is populated
w-usage Raised when an element with W usage is populated
re-usage Raised when an element with RE usage is not populated
o-usage Raised when an element with O usage is populated
cardinality The cardinality detection is issued when the number of repetitions of an element is not in the specified cardinality range.
null-cardinality The null cardinality detection is issued when a null field has more than one instance.
length-spec-error-no-valid No valid length specification was found
length-spec-error-xor Length specification error. One of (MaxLength, MinLength) or ConfLength should be present, but not both
length The length detection is issued when the length of a primitive element is not in the specified length range.
format The format detection is issued when the value of a primitive element is not valid according to the format defined or derived from the data type.
extra This detection is issued when a complex element has extra children
unescaped This detection is issued when a primitive element contains unescaped separators
unexpected This detection is issued when an unexpected segment is found at a specific position
invalid This detection is issued when an invalid content is found in the message
unresolved-field This detection is issued when a the reference values are not found in dynamic mapping for varies fields
Content based detections
coconstraint-failure This detection is issued for a failed coconstraint verification
coconstraint-success This detection is issued for a successful coconstraint verification
highlevel-content Order Indifferent trigger not found
constraint-failure This detection is issued for a failed constraint verification
constraint-success This detection is issued for a successful constraint verification
constraint-spec-error This detection is issued for a invalid constraint verification
content-failure This detection is issued for a failed content verification
content-success This detection is issued for a successful content verification
content-spec-error This detection is issued for a invalid content check verification
predicate-success This detection is issued for a successful predicate verification
predicate-failure This detection is issued for a failed predicate verification
predicate-spec-error This detection is issued for a invalid predicate verification
evs This detection is raised when the code is in the value set but the usage is E (excluded)
pvs This detection is raised when the code is in the value set but the usage is P (permitted)
code-not-found This detection is raised when the code cannot be found in the value set
vs-not-found This detection is raised when the value set cannot be found in the library
empty-vs This detection is raised when the value set is empty
vs-error This detection is raised when there is an error in the value set
binding-location This detection is raised when there is an error while resolving the binding location
vs-no-validation This detection is raised when the value set is excluded from the validation
coded-element This detection is raised when coded element validation fails
